# Bank Marketing Dashboard

## Project Overview
This project analyzes a bank marketing campaign dataset using Power BI.

## Key Metrics
- Total Customers: 45K
- Total Subscribers: 5K
- Conversion Rate: 11.70%
- Average Balance: 1.36K

## Dashboard Insights
- Subscription rate by age
- Subscription rate by education
- Subscription rate by housing loan
- Campaign performance analysis
- Contact type analysis
- Monthly subscription trends

## Tools Used
- Power BI
- Excel/CSV
- DAX
